Данная программа реализует шифрование методом Цезаря (http://ru.wikipedia.org/wiki/Шифр_Цезаря).
Вы можете использовать данный код на ваше усмотрение!
На данный момент в программе присутствует:

* Шифровка/расшифровка английского текста (остальные символы при кодировании игнорируются)
* GUI
